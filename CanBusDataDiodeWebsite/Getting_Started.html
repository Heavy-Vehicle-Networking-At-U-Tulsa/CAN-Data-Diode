Getting Started with the Data Diode:

Instructions are written for a Windows machine. All program downloads and software requirements will be listed in the downloads section of this website, along with a printable copy of the following instructions.

The following instructions are written with the assumption that you have received a data diode testing kit. If you do not have a kit, please submit an application.

Download Github Repository

The device will function properly as delivered. It will not allow messages to be transmitted from one side to the other; however, to optimize the functionality of the device, it is necessary to program the two on board micro-controllers. Currently, the following github repository has provisions for sending request messages on the J1939 side of the network, with example code for requesting VIN and Engine Hours.
